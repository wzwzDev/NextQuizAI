## How to Update URLs

1. **Upload screenshots to GitHub**:
   - Go to your repository on GitHub
   - Create an issue or go to an existing one
   - Drag and drop each screenshot into the issue comment box
   - GitHub will generate URLs like: `https://github.com/user-attachments/assets/[unique-id].png`

2. **Update README.md**:
   - Replace the placeholder URLs with the actual GitHub-generated URLs
   - Maintain the same order and descriptions

3. **Alternative - Use repository screenshots folder**:
   - Upload images to `/screenshots/` directory in your repo
   - Use relative paths like: `screenshots/homepage.png`
   - This keeps images in your repository instead of as attachments

## Example GitHub Upload Process

1. Go to: https://github.com/wzwzDev/TFM/issues
2. Create a new issue titled "Screenshots for README"
3. Drag each screenshot into the comment area
4. Copy the generated URLs
5. Update the README.md file with the real URLs
6. Commit and push the changes

Your README is now properly structured and ready for the actual screenshot URLs!
